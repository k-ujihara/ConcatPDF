説明

PDF を操作するアプリケーションです。次の操作ができます。

PDF の結合 
PDF の一部ページの抽出 
PDF の暗号化
PDF のビューの設定
PDF の暗号化解除 （DecryptPDF）
PDF の1ページごとへの分割 （SplitPDF）
画像ファイルのPDFへの変換

CUI バージョンと GUI バージョンがあります。

インストール方法

　ConcatPDF-???.zip の中の Setup.exe を実行する。
